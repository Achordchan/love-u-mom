## ✨ 特性

- 📱 拼命适配iPhone 11 Pro Max
- 🎮 适合老妈的休闲游戏
- 🔍 支持中文/拼音/首字母快速搜索
- 💾 本地记录最近游戏，快速访问
- 🌙 柔和界面，护眼配色
- 📲 支持 iOS 全屏应用模式
- 💬 一键反馈游戏需求，接入wxpusher。

## 🛠️ 技术栈

- HTML5 + CSS3
- 原生 JavaScript
- [pinyin-pro](https://github.com/zh-lx/pinyin-pro) - 拼音搜索支持
- [WxPusher](https://wxpusher.zjiecode.com/docs) - 微信消息推送
